# ATM Simulation in Java

A simple console-based ATM simulation program made using Java.

## Features
- Login using PIN  
- Check account balance  
- Deposit money  
- Withdraw money  
- Exit system  

## Technologies Used
- Java  
- Scanner class  
- Loops, if-else, switch-case
